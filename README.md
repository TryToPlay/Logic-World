# Logic-World
In my second year of college our Teacher introduced to a digital circuit simulator called logisim.
This project is inspired by it.

Controls
Press T to toggle between modes.
In Arrangement Mode- (Arrow Cursor)
Up and Down arrow keys to switch between componeent type (input, gate and output).
Left and Right arrow keys to switch between components.
Single click to place component, double click to remove component.
In Connections Mode- (Crosshair Cursor)
Click on two components in succession to connect them. By default, if two gates are connected, the output of left gate becomes input for right gate.
In Interaction Mode- (Pointer Cursor)
Click on interactive components (like switches) to change their state.
